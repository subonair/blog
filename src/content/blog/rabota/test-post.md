# Асинхронная работа без хаоса

Команда из 8 человек в 5 часовых поясах. Никаких обязательных встреч. Проект едет.

## Три правила

**1. Письменно всё**
Решение принято? Напиши. Обсудили голосом? Напиши саммари.

**2. Статус без вопросов**
Каждый день — короткий апдейт: что сделано, что дальше, есть ли блокер.

**3. Доверие по умолчанию**
Не контролируй часы — контролируй результат.

## Инструменты

Linear + Notion + Loom. Минимум звонков, максимум контекста.

Работает. Проверено.
